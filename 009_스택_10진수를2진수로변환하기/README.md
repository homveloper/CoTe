## 다른 응용 방안

- 8진수, 16진수 등 다른 진법 변환에도 동일한 스택 기반 접근법 적용 가능

```Python
def decimalToString(n, base):
    """10진수 n을 base 진수 문자열로 변환"""
    digits = []
    while n > 0:
        digits.append(str(n % base))
        n //= base
    # 스택에서 값을 꺼내어 문자열 구성
    result = ''
    while digits:
        result += digits.pop()
    return result
```

- 스택을 활용한 후위 표기법 계산기 구현에도 응용 가능
