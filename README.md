## 사용 방법

1. `_template` 폴더를 복사하고 번호_주제_문제명 형식으로 이름 변경
2. README.md에 문제명, 해시태그, 풀이 과정 및 회고 작성
3. solution.py에 풀이 코드 작성
4. 메인 README.md 테이블에 추가

```bash
# 템플릿 폴더 복사 (번호, 주제, 문제명 입력)
cp -r _template 001_해시_두개뽑아서더하기
```

**예시**
- `001_해시_두개뽑아서더하기/`
- `002_해시_완주하지못한선수/`
- `003_스택_올바른괄호/`
- `004_정렬_가장큰수/`
