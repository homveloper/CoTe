# [두개뽑아서더하기]

#배열 #집합 #순회

## 풀이 과정

### 핵심 아이디어
- 중복되지 않은 결과를 원하므로, set에 데이터를 보관
- sorted는  iterable한 데이터를 list로 변환해줌

### 접근 방법
1. 빈 set을 생성
2. 동일하지 않은 두 수를 선택하는 모든 경우의 수를 반복
3. 두 수를 더한 결과를 set에 저장
4. set을 sorted로 정렬

## 회고
