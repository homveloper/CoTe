#구현 #방향 #중복처리

## 풀이 과정

### 핵심 아이디어
- 이동 중복 여부는 set 자료구조를 활용하여 처리
- set은 유니크하므로  set의 길이로 중복 없는 경로 수 파악 가능

### 접근 방법
1. 방향 순회
2. 이동 가능 여부 확인
3. 이동 경로를 set에 추가
   1. 이동 경로는 방향이 없으므로, (출발점, 도착점), (도착점, 출발점) 모두 추가
4. 최종적으로 set의 길이 반환
5. 이동 경로는 왕복이므로 2로 나누기

## 회고

### 배운 점
- set 자료구조의 유용성
- 함수에서 tuple 반환 및 언패킹 활용법

### 어려웠던 부분
- 이동 경로를 어떻게 표현할지 고민
